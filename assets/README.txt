AL-BAZ GROUP — IMAGE ASSETS
===========================

Drop each company's photos into its folder, named 1.jpg, 2.jpg, 3.jpg ...

  assets/foods/      -> Baz Ready-Made Foods
  assets/tobacco/    -> Al-Baz Molasses Tobacco
  assets/visions/    -> Modern Visions Trading
  assets/tayeb/      -> Tayeb Al-Khair Trading & Real Estate
  assets/beethoven/  -> Beethoven Restaurant & Café

The gallery for each project reads its list from script.js (the
galleryData object near the top). To add or remove images, edit that
list — add lines like:
    { src: 'assets/foods/4.jpg', alt: 'New line', altAr: 'خط جديد' }
The dots, arrows, and autoplay adjust automatically.

Until you add the real files, each slide shows a blue placeholder with
a label, so the site looks intact while you collect photos.

Recommended: landscape images, ~1600x1000px, JPG or WebP.
